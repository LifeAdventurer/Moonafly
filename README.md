# Moonafly

## Secret files

1. Set your token via `config.json`:

    ```json
    {
        "token": "{your TOKEN}"
    }
    ```

2. Set your user identity list via `user_identity.json` including author, developer, guests role:

    ```json
    {
        "author": [
            
        ],
        "developers": [

        ],
        "guests": [

        ]
    }
    ```

## Run Moonafly

### Windows

```bat
run_moonafly.bat
```

### macOS/Linux

```bash
run_moonafly.sh
```
